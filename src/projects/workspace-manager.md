Workspace Manager is a cloud development environment platform that lets teams spin up fully configured development desktops in seconds. It combines a React frontend with a Go backend, AWS infrastructure (ECS, DynamoDB, S3), and WebRTC-based real-time desktop streaming.

Key features include automated VM provisioning, role-based access control, real-time collaboration, and a mock development API for local testing without cloud dependencies.
