Dexbox is a Go-based agent that automates the entire lifecycle of virtual machines — from creating custom installation ISOs with injected credentials and configurations, to provisioning VMs via VirtualBox, to applying automated security hardening.

It supports custom username/password injection, unattended Windows installation, and a comprehensive hardening script that disables unnecessary services and configures firewall rules.
